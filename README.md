# 📓 Project Notebook

Project Notebook is a sleek, premium, and fully native macOS application built with **SwiftUI**. Designed specifically for developers and creators, it provides a highly organized, modular workspace to manage project logs, version histories, and feature documentations—all with a VS Code-inspired productivity flow.

## ✨ Features

- **🏠 Multi-Level Organization:** Manage Projects → Versions → Features (.txt files) in a clean, hierarchical structure.
- **🎨 Modern macOS Design:** A smooth and native interface with auto-adaptive Dark and Light modes.
- **⚡ VS Code-Inspired UI:** Productivity-first column layout with a dynamic tab bar for feature switching.
- **📁 File-Based Persistence:** No heavy databases. Everything is stored directly in your local file system for privacy and portability.
- **🌐 Multi-Language Support:** Seamlessly switch between **English** and **Turkish** with a stylish UI toggle.
- **🧼 Premium Experience:** Custom-built vector app logo, smooth transitions, and refined SF Symbols integration.

---

## 🛠 Tech Stack

- **Framework:** SwiftUI (Native macOS)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Data:** FileManager API (Disk-based storage)
- **Style:** SF Symbols 5 & Hierarchical Rendering

---

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sairiahzan/Project-Notebook.git
   ```
2. Open `Project Notebook.xcodeproj` in **Xcode**.
3. Build and Run (**⌘ + R**).
